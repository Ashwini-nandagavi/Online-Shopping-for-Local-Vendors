Online Shopping for Local Vendors

Developed a web application that connects local vendors with customers for online shopping. The system allows users to browse, order, and purchase local products easily while helping vendors manage their listings and sales online.

The Online Shopping For Local Vendor’s project is consists of following modules and Forms. 

1.	Admin Module : In Admin module forms are
    •  Login->Dashboard
    • View/Delete Shop Seller
    • View/Delete Customers 
    • View Review
    • View Queries 
    • Change Password 
    • Logout 

  2 . Seller Module : In seller Module Forms are
   • Sing Up ->Sing In-> Dashboard 
   • Profile 
   • Add/View  Products
   • Orders  
   • Sells Report
   • View Review
   • Change password
   • Logout 

3. Customer Module : In Customer module forms are 
  • Sign Up –> Log In 
  • Customer Profile 
  •  Product  
  •  My  Orders
  •  Review
  •  Change password 
  •  Logout
